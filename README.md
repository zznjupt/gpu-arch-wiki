# GPU 微架构总结 v2.3

> 一个系统整理 GPU 微架构、计算单元、内存层次、数据格式等知识的个人知识库。

📊 **当前模块**: 数据精度格式知识图谱

🔢 **16种格式** · 💻 **5大厂商** · 🏗️ **16个硬件系列** · ⚡ **纯静态** · 🚀 **持续扩展**

## 项目定位

这是一个**个人 GPU 知识库**，用于系统整理和归纳 GPU 相关的各种知识。

### 📚 已完成模块

#### 🔢 数据精度格式知识图谱
- 16 种主流数据精度格式（FP64/32/16, BF16, FP8, INT8/4...）
- 8 个核心概念卡片（位宽、精度、动态范围等）
- 位结构可视化（三色图解）
- 16 个硬件系列支持矩阵
- 31 个多框架代码示例
- 点击卡片查看详细模态框

### 🚧 规划中模块

#### 🧮 计算单元
- Tensor Core 架构演进
- CUDA Core 工作原理
- 计算吞吐对比

#### 💾 内存架构
- 显存层次结构
- L1/L2 缓存机制
- 内存带宽分析

#### ⚡ 性能优化
- 性能调优技巧
- Benchmark 方法
- 瓶颈分析

（更多模块持续添加中...）

### 核心功能

✅ **位结构可视化** - 三色图解（🔵符号/🟢指数/🔴尾数），直观理解格式

## 📊 已收录内容

**16种精度格式**（按分类排序）:
- 标准浮点: FP64, FP32, FP16
- AI 专用: TF32, BF16, FP8 E4M3/E5M2
- 量化格式: INT16/8/4/1, FP4, NF4, MXFP4
- 扩展格式: Posit, Block FP

**5大硬件厂商**:
- NVIDIA: Volta, Turing, Ampere, Hopper, Blackwell (5代)
- AMD: CDNA 2, CDNA 3 (2代)
- Intel: Xe, Gaudi (2代)
- Google: TPU v3, v4, v5e, v5p (4代)
- 华为: 昇腾 910, 910B, 910C (3代)

## 💡 使用提示

### 位结构可视化 🎨
- **卡片预览**: 三色条显示位分配（🔵符号/🟢指数/🔴尾数）
- **详细图解**: 点击卡片查看完整位结构图（类似 Float_example.png）
- **快速对比**: 比较 FP32 vs BF16 的指数和尾数差异

### 查看代码示例
点击格式卡片（如 BF16），在模态框中查看 PyTorch/TensorFlow 代码示例。

### 硬件支持
在页面底部"硬件支持"区块查看矩阵（16 列），参考颜色图例：
- 🟢 原生支持（最佳性能）
- 🟠 部分支持
- ⚫ 不支持
