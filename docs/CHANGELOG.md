### 位结构可视化 ✨

#### 格式卡片预览
- 每个浮点格式卡片添加位结构预览条
- 三色编码：
  - 🔵 蓝色 - 符号位
  - 🟢 绿色 - 指数位
  - 🔴 红色 - 尾数位
- 长度按实际位数比例显示
- 悬停显示位数说明

#### 模态框详细图解
- 完整的位结构 SVG 图解（**居中对齐**）
- 显示每个位的 0/1 值（示例）
- 标注位索引和位段名称
- 类似 Float_example.png 的效果
- 自适应布局策略：
  - FP64 (64位): 15px/位，总宽 1000px，位图居中
  - FP32 (32位): 24px/位，总宽 800px，位图居中
  - FP16/BF16 (16位): 38px/位，总宽 750px，位图居中
  - FP8 (8位): 70px/位，总宽 650px，位图居中
- Posit 和 Block FP 显示"位结构可变"说明（不显示位图）
- **修复**: 所有位图通过计算偏移量实现居中显示

### 格式卡片排序优化

#### 按分类组织
格式卡片现在按类别有序排列：

1. **标准浮点** (3个): FP64 → FP32 → FP16
2. **AI 专用** (4个): TF32 → BF16 → FP8 E4M3 → FP8 E5M2
3. **量化格式** (7个): INT16 → INT8 → INT4 → INT1 → FP4 → NF4 → MXFP4
4. **扩展格式** (2个): Posit → Block FP

**优势**:
- 同类格式连续显示
- INT 系列集中在一起
- FP 系列分类清晰
- 浏览更有逻辑性

## v2.0 - 2026-04-10

### 新增功能

#### 1. 格式详情模态框 ✨
- **点击格式卡片**即可弹出详细信息模态框
- 展示完整的位结构（符号位、指数位、尾数位）
- 显示内存占用和相对性能数据
- 优缺点对比展示
- **实用代码示例**（PyTorch/TensorFlow）
- 硬件支持芯片展示

#### 2. 新增数据维度 📊
为每个格式添加：
- `memoryPerParam`: 每参数内存占用（字节）
- `relativeSpeed`: 相对性能（以 FP32 为基准 1.0）
- `codeExample`: 实用代码示例

#### 3. 新增可视化图表 📈

**内存占用对比图**
- 横向柱状图对比各格式的内存需求
- 悬停显示 10B 参数模型的实际内存占用
- 按内存大小排序，颜色编码

**相对性能对比图**
- 展示各格式相对于 FP32 的性能提升
- 悬停显示性能提升百分比
- INT4 和 FP8 等量化格式性能优势明显

#### 4. 代码示例库 💻
为主要格式添加实用代码示例：
- **FP32/FP64**: 基础使用
- **FP16**: 混合精度训练（含 GradScaler）
- **BF16**: autocast 使用
- **TF32**: NVIDIA GPU 自动加速配置
- **FP8**: Transformer Engine 使用
- **INT8**: 动态量化示例
- **INT4/NF4**: GPTQ/AWQ/QLoRA 量化
