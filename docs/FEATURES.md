## 4. 代码示例库 💻

### 覆盖场景

#### 混合精度训练
```python
# FP16 + GradScaler
from torch.cuda.amp import autocast, GradScaler
scaler = GradScaler()
with autocast():
    loss = model(input)
scaler.scale(loss).backward()
```

#### GPU 加速
```python
# TF32 自动启用 (Ampere+)
torch.backends.cuda.matmul.allow_tf32 = True
torch.backends.cudnn.allow_tf32 = True
```

#### 量化部署
```python
# INT8 动态量化
model_int8 = quantization.quantize_dynamic(
    model, {torch.nn.Linear}, dtype=torch.qint8
)
```

#### LLM 量化
```python
# NF4 QLoRA
bnb_config = BitsAndBytesConfig(
    load_in_4bit=True,
    bnb_4bit_quant_type="nf4"
)
```

#### FP8 训练
```python
# Transformer Engine (H100)
import transformer_engine.pytorch as te
with te.fp8_autocast(enabled=True):
    output = model(input)
```

## 5. 硬件支持矩阵 💻

### 支持级别（带图例）
矩阵上方显示颜色图例：
- 🟢 **原生支持** (native): Tensor Core 硬件加速，最佳性能
- 🟢 **完整支持** (full): 软硬件完整支持
- 🟠 **部分支持** (partial): 有限支持或模拟实现
- ⚫ **不支持** (none): 无支持

### 快速查询
- **按格式**: 查看某格式在哪些硬件上可用
- **按硬件**: 查看某硬件支持哪些格式

### 实用案例
```
问：我有 A100 GPU，推理用什么格式最快？
答：查看 Ampere 列 → BF16、TF32、INT8 都是原生支持

问：我想用 FP8，需要什么硬件？
答：查看 FP8 行 → Hopper (H100)、Blackwell、CDNA3
```

## 7. 核心概念卡片 📚

### 8 个关键概念
整合在格式总览页面，包括：

1. **位宽 (Bit Width)** - 存储和带宽需求
2. **精度 (Precision)** - 数值表示能力
3. **动态范围 (Dynamic Range)** - 最大最小值范围
4. **混合精度 (Mixed Precision)** - 多格式协同
5. **指数位 (Exponent)** - 动态范围控制
6. **尾数位 (Mantissa)** - 精度控制
7. **量化 (Quantization)** - 模型压缩技术
8. **IEEE 754** - 标准浮点格式规范
