# v2.1 更新说明

## 主要变化

### 🎯 界面简化

**之前** (6 个板块):
- 格式总览
- 可视化对比
- 格式详情
- 硬件支持矩阵
- 选型指南
- 术语表

**现在** (3 个板块):
- 📊 格式总览 (整合了硬件矩阵、术语表)
- 📈 可视化对比
- 🔢 格式详情

### ✨ 新增功能

1. **硬件支持矩阵图例**
   - 位置：矩阵上方
   - 内容：
     - 🟢 原生支持（硬件加速）
     - 🟢 完整支持
     - 🟠 部分支持
     - ⚫ 不支持

2. **核心概念扩充**
   - 从 4 个扩展到 8 个
   - 整合了原术语表内容
   - 新增：指数位、尾数位、量化、IEEE 754

### 🗑️ 删除内容

- **选型指南板块** - 信息已包含在：
  - 格式详情模态框的"典型用途"
  - 格式卡片的用途标签
  - 可视化图表的性能对比

- **独立术语表** - 整合到格式总览的核心概念

### 📈 优势

#### 信息更集中
- 一个页面查看核心概念 + 硬件矩阵
- 减少页面跳转次数
- 提高信息获取效率

#### 界面更清爽
- 侧边栏导航项从 6 个减少到 3 个
- 更简洁的信息架构
- 更符合"知识图谱"定位

#### 更易理解
- 硬件矩阵有清晰的图例说明
- 核心概念就近展示
- 降低学习曲线

## 使用建议

### 第一次访问
1. 从"格式总览"开始
2. 阅读核心概念卡片
3. 查看硬件支持矩阵（参考图例）
4. 切换到"格式详情"浏览具体格式

### 日常使用
1. 直接进入"格式详情"
2. 使用筛选器快速定位
3. 点击卡片查看代码示例
4. 在"可视化对比"查看性能数据

## 技术改进

- ✅ 更少的 DOM 节点
- ✅ 更简单的导航逻辑
- ✅ 更好的信息密度
- ✅ 保持所有核心功能

## 迁移指南

如果你之前使用过 v2.0：

| v2.0 位置 | v2.1 位置 |
|----------|----------|
| 术语表 → Exponent | 格式总览 → 核心概念 → 指数位 |
| 术语表 → Mantissa | 格式总览 → 核心概念 → 尾数位 |
| 术语表 → 其他 | 格式总览 → 核心概念 |
| 硬件支持矩阵 | 格式总览 → 硬件支持矩阵（带图例） |
| 选型指南 → 训练推荐 | 格式详情 → BF16 模态框 |
| 选型指南 → 推理推荐 | 格式详情 → INT8/FP16 模态框 |

---

**总结**: v2.1 是一次"减法"优化，通过整合和精简，让页面更简洁、更高效，同时保留了所有核心功能。
